Программа состоит из ассемблерной и C-части. 
Ассемблер настраивает обработчик прерываний, 
таймер и переходит в C-код. C-код через системные вызовы 
(ecall) запрашивает доступ в M-режим (печать, работа с таймером).


Ассемблерные блоки:

Дефайны: разрядность (32/64 бита), адреса CLINT (mtime, mtimecmp).

_start: настройка PMP, загрузка trap_vector в mtvec, включение прерываний в mstatus, переход в main.

timer_setup: запись в mtimecmp = текущее mtime + интервал (если 0 — максимум, отключение).

timer_init: разрешение таймерных прерываний в mie, установка mtimecmp в максимум.

trap_vector: сохранение регистров, вызов handle_trap(cause, mepc, sp), восстановление mepc, регистров, mret.

park: ожидание прерываний (wfi) для остальных ядер.

end: бесконечный цикл.


C-функции:
print, timer_setup, timer_init: работают в M-режиме (вывод в UART, управление таймером).

syscalls: кладёт аргументы в a0–a7, вызывает ecall для перехода в M-режим.

syscall_print / timer_setup / timer_init: обёртки, вызывающие syscalls с нужным типом (1,2,3).

handle_trap:

Прерывание (cause < 0): если code=7: таймер, выключить его.

Исключение (cause >= 0): если code=8 (ecall): по regs[15] определить тип (1,2,3) и вызвать соответствующую M-функцию. Для исключений epc += 4.

main: последовательные вызовы: печать, инициализация таймера, установка интервала, wfi, end.
